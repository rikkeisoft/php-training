Money lover


1. Environments:
   1. OS: Windows 7/ Linux
   2. IDE: Netbeans, Notepad++
   3. Web Server: XAMPP / WAMP / Apache 2.2+
   4. PHP: 5.3+
   5. SQL Server: MySQL
   6. Framework: CakePHP 2.4.x, Twitter Bootstrap 3
   7. Coding style: http://www.php-fig.org/psr/psr-2/



2. Functions list:
      1. Accounts & Authentication
         * Register, activate via email
         * Update profile
         * Change / Forgot password
         * Login / Logout
    
      2. Wallets management
         * Add wallet
         * Update wallet info
         * Set default wallet
         * Change current wallet
         * Transfer money between wallets
    
      3. Categories management
         * Add / Edit categories
         * Note: 
            * Logic for special categories (loan, borrowings, ...)
            * Categories have belong “spent on” or “earned from”
         * Delete category

      4. Transaction
         * Add / Edit transaction
            (Note: With loan and borrowing, add return/pay money option)
         * Delete transaction
         * Show transactions by date time, categories
         * Monthly report
